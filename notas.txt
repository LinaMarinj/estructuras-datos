Estructuras de datos (Java script)

1 - Arreglos (vectores)
    Simulación de datos (almacenamiento)

* Los arreglos no tiene tipo de dato en java script 
* Tamaño o longitud -> Dado por la cantidad de elementos dentro del mismo
* Posición -> Dado por la ecuación n-1, donde n es el elemento
* Los arreglos estan asociados al ciclo for 

--> Recorrido de un arreglo -> Es la iteración dentro del arreglo por cada
elemento: buscar contenido y relaizar alguna operación matemática o booleana


--> llenado de un arreglo: Es la iteración dentro del arreglo por cada
elemento: agrega contenido a una posición del arreglo

Métodos de los arreglos:

-push(), pop(), shift(), unshift(),
- map(), find(), foreach(), filter(), some(), join(), split()


* push(): método que agrega un elemento a la última posición
* pop(): método que elimina la última posición
* shift(): método que elimina la primera posición
* unshift(): método que agrega un elemento a la primera posición
* splice((empieza y elimina)posicion y cantidad de elementos): método para eliminar un 
elemento seleccionado.

* map() -> método que recorre un arreglo, aplica una condición y retorna 
un nuevo arreglo con la nueva información. El método map es un método no mutable, 
significa que no cambia el valor original de arreglo



*find() -> método que recorre un arreglo, aplica una condición y retorna un elemento. El método filter es no mutable, significa que no cambia el valor original del arreglo
